# Pharmaceutical Sales Dashboard – Power BI

This project presents an interactive Power BI dashboard designed to analyze pharmaceutical sales performance across Germany and Poland.

## Dashboard Overview

The dashboard provides insights into:

- Total sales performance
- Sales trends
- Product performance
- City-level sales distribution
- KPI monitoring
